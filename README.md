# Tips Analysis Dashboard (Power BI)

This project presents an interactive dashboard built using Power BI to analyze customer tipping behavior.

## Features:
- Total Bill and Total Tip KPIs
- Tips by Day (Bar Chart)
- Tips by Gender (Pie Chart)
- Tips by Time (Pie Chart)

## Tools:
- Power BI Desktop# tips-analysis-dashboard
